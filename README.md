Feedstock for [emela](https://github.com/gstagnit/eMELA) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

eMELA: electroweak parton distributions and matched EW-QCD calculations

## Architectures

| Architecture | Latest published |
|--------------|------------------|
| linux-amd64 (`linux-64`) | ✅ `1.0` |
| linux-arm64 (`linux-aarch64`) | ✅ `1.0` |

_As of the last feedstock render; the badges above are live._


## Install

```bash
conda install -c hep-forge -c conda-forge emela
```
